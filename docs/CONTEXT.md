# docs/ — Context

User- and developer-facing documentation for Reactable Wall.

- `api/` — module/param reference: each marker ID, its module, and what rotation
  controls (mirrors `src/services/moduleRegistry.js`).
- `guides/` — install & calibration guide (camera placement, anti-glare matte surface,
  short-throw projector mounting, running auto-calibration). Write during Phase 3.
- `changelog/` — notable changes per phase.

Keep docs in sync with `planning/specs/reactable-wall-design.md` — the spec is the
source of truth until the app ships.
