REACTABLE WALL

Place ID 0 (Oscillator) anywhere — it flows to the glowing center and sounds.
Add ID 1 (Filter) / ID 2 (Delay) near the path to the center to shape it.
ID 6 (Sequencer) near an oscillator adds rhythm; ID 5 (Tonality) makes it melodic.
ID 4 (LFO) modulates the nearest puck; ID 3 (Volume) sets master level.

M: toggle markers  |  [ ]: marker size  |  C: calibration  |  W: webcam
